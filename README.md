# distributed-restaurant-app

## Build a Web Page with React and NodeJS/Express
Rewrite/Enhance the developed food website using React, Node.js and Express. The server
directory hierarchy is the same as b4. Rewrite JavaScript into React and merge separation of
structure, presentation, behavior files into component style (i.e., components contain all their
structure, presentation and behavior).


## Steps to run application: -


Open Terminal 1
>>cd server
>>npm run dev
Open Terminal 2
>>cd client
>>npm start
